클라우드 컴퓨팅(Cloud Computing)

2019-12-18
AWS

들어가기 전에

  • 아마존 웹 서비스(AWS)를 공부하기에 앞서 알아야 할 것은 클라우드 컴퓨팅 이라는 기술이다.
  • AWS 도 결국 클라우딩 컴퓨팅 기술을 기업이 사용 할 수 있도록 제공하는 서비스 이기 때문에 먼저 이 클라우딩 컴퓨팅이 뭔지 부터 알아야 한다.

Cloud Computing

: IT 리소스를 물리적 데이터 센터나 서버가 아닌 인터넷을 통해 온디맨드로 제공하고 사용한 만큼만 비용을 지불하는 것

그럼 제공은 대체 누가 해?

: 대표적인 클라우드 공급자인 아마존 웹 서비스

Amazon Web Services(AWS)와 같은 클라우드 공급자로부터 필요에 따라 컴퓨팅 파워, 스토리지, 데이터베이스와 같은 기술 서비스에 액세스할 수 있습니다.

누가 써?

  • 서비스가 크지 않지만 기업으로 운영하는 경우, 비용을 지불하고 쓰기에 합리적인 것 같다.
  • 서비스가 크면 물리적인 서버가 필요해지고 관리할 인력도 추가로 필요해진다.
  • 이러한 이유로 리소스가 적은 많은 스타트업들이 사용 중이다.
