Web Crawling (2) BeautifulSoup

2019-09-02
python

사용 배경

웹 크롤링을 하기 위해서는 기본적으로 웹 페이지의 도메인에 요청해야 합니다.

파이썬의 request 모듈로도 가능하지만, 이를 통해 가져온 데이터는 단순한 문자열 형태이기 때문에 파싱(Parsing)하고 데이터를 분석하기에는 한계가 있습니다.

BeautifulSoup

  • 웹 크롤링을 할 때 사용하는 Python 라이브러리로 HTML, XML 파일에서 원하는 데이터를 손쉽게 파싱 할 수 있도록 도와줍니다.

설치

  1. 파이썬 설치 명령(pip or pip3) 사용

  2. 파일 다운로드
    http://www.py4e.com/code3/bs4.zip 다운로드 후 폴더에 압축을 해제합니다.
