파이썬으로 웹 크롤링(Web Crawling) 하기(1) Regular Expression

2019-09-01
python, 정규식

정규식(Regular Expression)

  • 문자열 추출을 위해 작성하는 표현식
  • 정규식을 활용하여 얼마든지 원하는 패턴의 문자열을 추출해낼 수 있다.
  • Python도 정규식을 위한 모듈을 제공하고 있다.

핵심 정규식 표현

  • ^ : 문자열의 시작
  • $ : 문자열의 끝
  • . : 문자
  • \s : 공백
  • \S : 공백 없는 문자열
    • : 0 이상 반복
  • *? : 0 이상 반복 (non-greedy)
    • : 하나 이상 반복
  • +? : 하나 이상 반복 (non-greedy)
  • [aeiou] : 모음과 매칭
  • ( : 추출 시작점
  • ) : 추출 끝점

greedy vs non-greedy

  • *, + 의 경우 수량자를 탐욕적으로 설계하여서 가능한한 가장 큰 덩어리를 찾으려 한다.
  • 그러한 추출을 피하고 싶다면, 해당 문자(* or +)뒤에 ‘?’를 붙여주면 된다.

  • 예시

    1. ^F.+:
      ‘From: Using the :’
    2. ^F.+?:
      ‘From:’

내장 함수

  • re.search() : 정규식 패턴과 매칭되는 문자열을 반환한다.
  • re.findall() : 정규식 패턴과 매칭되는 문자를 리스트로 반환한다.(매칭되는 문자열이 없다면 빈 리스트 반환)
